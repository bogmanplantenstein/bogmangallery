Bogman Gallery v2

One-time migration from v1 localStorage → data.json

Step 1 — Detect v1 Data

This tool reads your old admin's localStorage and converts it to the new data.json format. Open this file in the same browser you used to run the old Bogman admin.

Keys it looks for:

  • bogman_taxa / bogman_admin_data — taxa array
  • bogman_care_defaults — group care defaults
  • bogman_config / bogman_admin_config — genus groups, habitats

Step 2 — Review & Options

If photos are Squarespace CDN URLs, they'll be prefixed with sq:.
If they're Google Drive file IDs, they'll be prefixed with drive:.
If mixed or unsure, choose "Auto-detect".

Step 3 — Download & Install

Your migrated data.json is ready. Download it and place it at:

F:\Apps\Bogman Gallery\v2\data\data.json

It will overwrite the empty template that was created during setup.
